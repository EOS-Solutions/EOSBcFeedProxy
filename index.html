EOS APPS NuGet Feeds

URL Format

All NuGet feed URLs follow a consistent structure to ensure easy access and integration:

https://feed.eos-solutions.app/{BC version}/{Type}/index.json
Available Feed Types

Release

Public access - no authentication required

Contains production-ready, stable versions of all EOS Apps for Business Central.

LS Retail

Secured access - MSAL authentication required

Contains third-party apps from LS Retail. Access restricted to authorized users.

Project365

Secured access - MSAL authentication required

Contains third-party apps from Kumavision for Project365. Access restricted to authorized users.

Example Usage

Here's how to construct a feed URL for Business Central version 26 Release packages:

https://feed.eos-solutions.app/26/Release/index.json

Additional Feed Types

Prerelease

Secured access - MSAL authentication required

Test

Secured access - MSAL authentication required

Contains Test apps for EOS Apps. Intended for internal use only.
